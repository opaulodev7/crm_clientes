# CRM Clientes

Sistema web completo de cadastro e gerenciamento de clientes desenvolvido com Flask + MySQL.

Projeto focado em:
- organização;
- simplicidade;
- visual profissional;
- CRUD completo;
- dashboard administrativa;
- portfólio para entrevistas.

---

# Funcionalidades

## Dashboard
- Total de clientes
- Total de serviços
- Faturamento total
- Últimos clientes
- Últimos serviços
- Gráfico com Chart.js

## Clientes
- Cadastro
- Listagem
- Busca por nome
- Edição via modal
- Exclusão
- Flash messages

## Serviços
- Cadastro
- Edição
- Exclusão
